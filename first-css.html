The "Class" Attribute

A Class is a name you give to a group of elements you want to style the same way.

<!-- HTML -->
<p class="highlight">Text 1</p>
<p class="highlight">Text 2</p>

/* CSS Syntax (starts with .) */
.highlight {
  color: green;
}

Result:

I have class="highlight"

I also have class="highlight"

The "ID" Attribute

An ID is a unique identifier for a single element.

<!-- HTML -->
<h1 id="main-title">Welcome</h1>

/* CSS Syntax (starts with #) */
#main-title {
  background-color: gold;
}

Result:

I am Unique (id="main-title")

Class vs. ID

Feature Class ID
HTML Attribute class="..." id="..."
CSS Symbol . (Dot) # (Hash/Pound)
Frequency Many elements Unique (ONE element)

The Box Model: Margin vs. Padding

Every element in HTML is essentially a box. Understanding the space inside and outside that box is crucial.

I am the Content

  • PADDING (Inside):
    The space between the Content and the Border.
    Like wearing a puffy winter coat. You get bigger, but it's attached to you.
  • MARGIN (Outside):
    The space between the Border and the Next Element.
    Like "Personal Space" or social distancing. It pushes others away.

Coding Space

How we write it in CSS:

.my-box {
  border: 1px solid black;
  /* Space INSIDE the box */
  padding: 20px;

  /* Space OUTSIDE the box */
  margin: 50px;
}
Pro Tip:
You can specify sides individually!

margin-top: 10px;
padding-left: 20px;
If you set margin: auto; with a width, the element centers itself horizontally!

The Display Property

This controls how elements behave in the "flow" of the page.
