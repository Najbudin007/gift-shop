What are HTML Forms?

HTML Forms are interactive elements that allow users to input and submit data to a web server or process it using JavaScript. Forms are essential for user interactions like registration, login, surveys, contact forms, and more.

Key Components:

  • <form> - Container element for all form controls
  • <input> - Various input types for different data
  • <textarea> - Multi-line text input
  • <select> - Dropdown menus
  • <button> - Submit or action buttons
  • <label> - Describes form inputs

Note about Storage: This demo uses in-memory storage (JavaScript variables) instead of browser storage. Data persists only during your current session and will be lost when you refresh the page.

Form Input Types

Text Input

<input type="text"> - Single-line text input for names, titles, etc.

Email Input

<input type="email"> - Validates email format automatically.

Password Input

<input type="password"> - Masks characters for security.

Number Input

<input type="number"> - Only accepts numeric values.

Date Input

<input type="date"> - Provides a date picker.

Textarea

<textarea> - Multi-line text input for comments, descriptions, etc.

Select (Dropdown)

<select> - Dropdown menu with predefined options.

Radio Buttons

<input type="radio"> - Single selection from multiple options.

Checkboxes

<input type="checkbox"> - Multiple selections allowed.
