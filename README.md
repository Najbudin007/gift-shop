# HTML, CSS & JavaScript Learning Repository

This repository was created for teaching students **HTML**, **CSS**, and **JavaScript** fundamentals.

## Overview

This project contains a collection of HTML files, stylesheets, and resources designed to help students learn and practice web development basics. It includes various examples and exercises covering:

- **HTML Structure**: Semantic markup, forms, and page layouts
- **CSS Styling**: Flexbox layouts, responsive design, and styling techniques
- **JavaScript**: Interactivity and DOM manipulation (as we expand)

## Project Structure

```
├── index.html              # Main landing page (Gift Shop)
├── basics.html             # Basic HTML concepts and examples
├── contact-form.html       # Form handling example
├── forms.html              # Form elements and validation
├── flexBox.html            # Flexbox layout demonstration
├── revision.html           # Revision exercises
├── background.html         # Background and styling examples
├── first-css.html          # Introduction to CSS
├── css/                    # Stylesheets
│   ├── style.css          # Main stylesheet
│   ├── home.css           # Home page styles
│   └── forms.css          # Form styles
└── img/                    # Image assets
```

## Getting Started

1. Clone or download this repository
2. Open any `.html` file in your web browser
3. Review the HTML structure and associated CSS files to understand how they work together

## Learning Topics

- HTML form creation and structure
- CSS Flexbox for layouts
- Responsive design principles
- CSS styling best practices
- HTML semantic elements

## Usage

These files serve as educational examples for:
- Classroom demonstrations
- Self-study reference material
- Practice exercises for beginners

Feel free to modify these examples and experiment with different HTML structures and CSS properties to deepen your understanding!
