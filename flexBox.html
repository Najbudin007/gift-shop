üéØ CSS Flexbox: Complete Guide

What is Flexbox?

Flexbox (Flexible Box Layout) is a one-dimensional layout method for arranging items in rows or columns. Items flex (expand) to fill additional space or shrink to fit into smaller spaces.

Why Learn Flexbox?

  • Easy Alignment: Center things vertically and horizontally with ease
  • Responsive Design: Items automatically adjust to screen size
  • No Floats: Cleaner, more predictable layouts
  • Space Distribution: Control spacing between elements intelligently

üåü Real-World Uses:

Navigation bars, card layouts, form layouts, toolbars, footer sections, mobile app interfaces, dashboard widgets

üì¶ Flexbox Basic Concept

Container vs Items

Flexbox has two key components:

  • Flex Container (Parent): Element with display: flex
  • Flex Items (Children): Direct children of the flex container

.container {
    display: flex;  /* This creates a flex container */
}

.item {
    /* These are flex items automatically */
}

Key Point: Once you add display: flex to a parent element, all direct children become flex items and line up in a row by default.

üìä Flex Item Properties

Control Individual Items

Flex Grow - Items Expand:

Flex Basis - Starting Size:

100px

.item {
    flex-grow: 1;      /* How much item grows (default: 0) */
    flex-shrink: 1;    /* How much item shrinks (default: 1) */
    flex-basis: auto;  /* Starting size (default: auto) */
    
    /* Shorthand: */
    flex: 1;           /* flex: grow shrink basis */
    flex: 1 0 200px;   /* grow 1, don't shrink, start at 200px */
}

Think of it: flex-grow is like "how greedy is this item for extra space?"

üìã Flexbox Cheat Sheet

Container Properties:

display: flex;
flex-direction: row | column;
justify-content: 
  flex-start | center | 
  flex-end | space-between |
  space-around | space-evenly;
align-items:
  stretch | flex-start |
  flex-end | center | baseline;
flex-wrap: nowrap | wrap;
gap: 20px;
align-content:
  flex-start | center |
  flex-end | space-between |
  space-around | stretch;

Item Properties:

flex-grow: 0;        /* How much to grow */
flex-shrink: 1;      /* How much to shrink */
flex-basis: auto;    /* Starting size */
flex: 1;             /* Shorthand: grow shrink basis */
align-self: auto;    /* Override container's align-items */
order: 0;            /* Change visual order */
