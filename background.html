Making Text Bold - HTML Way

There are two HTML tags to make text bold:

<!-- Using <strong> tag -->
<p>This is <strong>important</strong> text.</p>

<!-- Using <b> tag -->
<p>This is <b>bold</b> text.</p>

Result:

This is important text.

This is bold text.

Key Difference:
<strong> = Semantic (means important)
<b> = Visual only (just looks bold)

Making Text Bold - CSS Way

Use the font-weight property in CSS:

/* CSS */
.bold-text {
  font-weight: bold;
}

.extra-bold {
  font-weight: 900;
}

.light-text {
  font-weight: 300;
}

Font Weight Values:

Value Example
normal or 400 Normal Text
bold or 700 Bold Text
100-900 Extra Bold (900)

Making Text Italic - HTML Way

There are two HTML tags to make text italic:

<!-- Using <em> tag -->
<p>This is <em>emphasized</em> text.</p>

<!-- Using <i> tag -->
<p>This is <i>italic</i> text.</p>

Result:

This is emphasized text.

This is italic text.

Key Difference:
<em> = Semantic (means emphasis)
<i> = Visual only (just looks italic)

Making Text Italic - CSS Way

Use the font-style property in CSS:

/* CSS */
.italic-text {
  font-style: italic;
}

.oblique-text {
  font-style: oblique;
}

.normal-text {
  font-style: normal;
}

Font Style Values:

Value Example
normal Normal Text
italic Italic Text
oblique Oblique Text

HTML Tags vs CSS Properties

Style HTML Tags CSS Properties When to Use
Bold <strong>
<b> 		font-weight: bold; Use <strong> for important text
Use CSS for styling
Italic <em>
<i> 		font-style: italic; Use <em> for emphasis
Use CSS for styling
Best Practice:
Use semantic HTML tags (<strong>, <em>) when the text has meaning, and use CSS when it's just for visual styling!

Combining Bold and Italic

You can combine both styles in HTML or CSS:

<!-- HTML Way -->
<p><strong><em>Bold and Italic</em></strong></p>

/* CSS Way */
.bold-italic {
  font-weight: bold;
  font-style: italic;
}

Result:

This text is both BOLD and ITALIC!

Why Do We Need Background Images?

Reasons to Use Background Images:

  • Visual Appeal
    Makes websites more attractive and engaging
  • Branding
    Establish identity with hero sections, banners
  • Content Separation
    Create distinct sections on a page
  • Text Overlays
    Place text on top of images
  • Professional Look
    Modern websites use background images extensively

Background Image vs Regular <img> Tag

Feature Background Image (CSS) Regular <img> Tag
Purpose Decorative, backgrounds Content images (photos, diagrams)
Text Overlay ✓ Easy ✗ Difficult
Control Position ✓ Full control ~ Limited
Control Size ✓ background-size ~ width/height
SEO / Accessibility ~ No alt text ✓ Has alt text
Rule of Thumb:
Use background-image for decoration. Use <img> for content that matters!

Background Images - Inline CSS

Add background images directly in HTML using the style attribute:

<!-- Inline CSS -->
<div style="
  background-image: url('image.jpg');
  background-size: cover;
  background-position: center;
  padding: 40px;
  color: white;">
  <h2>Hello World</h2>
</div>

Background Images - External CSS

Better approach: Define background images in external CSS files or <style> tags:

/* External CSS or <style> tag */
.hero-section {
  background-image: url('hero.jpg');
  background-size: cover;
  background-position: center;
  padding: 60px;
  color: white;
}

<!-- HTML -->
<div class="hero-section">
  <h1>Welcome</h1>
</div>

Important Background Properties

Property Values Description
background-image url('path.jpg') Sets the background image
background-size cover, contain, 100px cover = fills entire area
contain = fits inside
background-position center, top left, 50% 50% Controls where the image is positioned
background-repeat no-repeat, repeat, repeat-x Controls if/how image repeats
background-attachment fixed, scroll fixed = stays while scrolling (parallax effect)
Tip: Always set background-size: cover; and background-position: center; for best results!

Pro Tip: Adding Dark Overlay

Make text readable on background images by adding a dark overlay using gradients:

/* CSS */
.overlay-section {
  background-image:
    /* Dark overlay */
    linear-gradient(
      rgba(0, 0, 0, 0.5),
      rgba(0, 0, 0, 0.5)
    ),
    /* Your image */
    url('image.jpg');
  background-size: cover;
}

Live Example:

Text is Now Readable!

The dark overlay makes text stand out

How it Works:
The rgba(0, 0, 0, 0.5) creates a semi-transparent black layer. Change 0.5 to control darkness (0 = transparent, 1 = fully black).

Summary

Font Styling:

  • Bold: <strong>, <b>, or font-weight: bold;
  • Italic: <em>, <i>, or font-style: italic;
  • Use semantic tags (<strong>, <em>) when meaning matters
  • Use CSS for pure visual styling

Background Images:

  • Use background-image: url('...');
  • Set background-size: cover; for best results
  • Add dark overlays with linear-gradient()
  • Prefer external CSS over inline styles
